# Student-Management-System
A SQL project to manage student records, courses, marks, and attendance using MySQL

# Overview
The Student Management System is a SQL-based database project developed using MySQL Workbench. It stores student, course, enrollment, marks and attendance informaton and demonstrates SQL queries for data retrieval and analysis.

# Features
- Students Record
- Course Management
- Attendance Tracking
- Marks Management
- Student Performance Analysis

# Database Tables
- Students
- Courses
- Enrollments
- Marks
- Attendance

# SQL Concepts Used
- Primary Keys
- Foriegn Keys
- INNER JOIN
- GROUP BY
- ORDER BY
- Aggregate Functions
- HAVING
- LIMIT

# Sample Queries
- Display all students
- Average marks by course
- Top 3 students
- Attendance analysis
- Student-course details using JOIN

# Tools Used
- MySQL Workbench
- SQL
- GitHub
